# Security Policy

> 脆弱性は **公開 Issue に書かないでください**（悪用を防ぐため）。

## 報告先
- 脆弱性・セキュリティ懸念は、公開 Issue ではなくメンテナへ**非公開で**報告してください。
- 受領後、可能な範囲で速やかに一次回答します。

## 対象バージョン
| Version | サポート |
|---------|---------|
| 0.1.x   | ✅ |
| < 0.1   | ✕ |

## 留意事項
- セッションログは平文保存（current-user ACL）。取り扱いに注意（経緯: UDR-amm-20260511T0803-a1c）。
- `.amm` の信頼境界・WebView2・IPC・設定書込の hardening は UDR-amm-20260608T1102-7f2 を参照。
