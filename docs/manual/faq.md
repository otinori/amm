# FAQ

利用者・運用者向けのよくある質問。

## Q. 起動時に CLI（Claude / Codex / Copilot）がすぐ終了する
A. 新規フォルダではセッション復帰トークンが無いため。コマンドごとの
「起動時にセッションを復帰する」設定は opt-in（既定 OFF）。
経緯: UDR-amm-20260610T0310-b6f / UDR-amm-20260610T0255-d8c。

## Q. 日本語入力（IME）で文字が二重に入る
A. 受信側のコアレッサ / dedup で緩和済み。再現する場合はログを添えて起票。
経緯: UDR-amm-20260615T0341-c5a 他。
